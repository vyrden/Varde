# Modules

Ce dossier accueille les modules officiels du projet à partir du
jalon 4 : `moderation`, `welcome`, `roles`, `logs`, `onboarding-presets`.

Aucun module ne réside dans `packages/core/`. Les modules officiels
utilisent exactement les mêmes API publiques que les modules tiers,
exposées par `@varde/core` et les autres packages.
